# ИКТИБ AI Platform

AI-ассистент для студентов ИКТИБ ЮФУ в Telegram. Студент задаёт вопрос — бот ищет ответ в верифицированной базе знаний института (стипендии, справки, общежития, пересдачи, контакты) и отвечает за несколько секунд. Технология RAG — нейросеть отвечает только на основе проверенных данных, не выдумывает.

**Что делаем:** Telegram-бот + Mini App (веб-интерфейс внутри Telegram) + панель администратора.  
**Стек:** NestJS, Next.js, PostgreSQL, Redis, RabbitMQ, Docker.  
**Срок:** 8 месяцев.
